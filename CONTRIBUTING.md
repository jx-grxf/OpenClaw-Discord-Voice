# Contributing

Thanks for your interest in contributing.

## Setup

Follow `docs/INSTALLATION.md`.

## Development flow

1. Create feature branch
2. Make changes in `src/`
3. Run checks:
   ```bash
   npm run build
   ```
4. Open PR with clear summary and test notes

## Guidelines

- Keep changes focused
- Avoid committing generated files or secrets
- Document behavior changes in README/docs
- Prefer small, reviewable commits
